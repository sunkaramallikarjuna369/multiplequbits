Exercise 1: Tensor Product Calculation

Problem: Calculate the tensor product |+⟩ ⊗ |1⟩ where |+⟩ = (|0⟩ + |1⟩)/√2

Solution:

|+⟩ ⊗ |1⟩ = (|0⟩ + |1⟩)/√2 ⊗ |1⟩
= (|0⟩ ⊗ |1⟩ + |1⟩ ⊗ |1⟩)/√2
= (|01⟩ + |11⟩)/√2

As a vector: [0, 1/√2, 0, 1/√2]T

import numpy as np
ket_plus = np.array([1, 1]) / np.sqrt(2)
ket_1 = np.array([0, 1])
result = np.kron(ket_plus, ket_1)
print(f"Result: {result}")
# Output: [0.  0.707  0.  0.707]

Exercise 2: Separability Test

Problem: Is the state |ψ⟩ = (|00⟩ + |01⟩ + |10⟩ + |11⟩)/2 entangled or separable?

Solution:

Let's try to factor this state:

|ψ⟩ = (|00⟩ + |01⟩ + |10⟩ + |11⟩)/2

We can rewrite this as:

= (|0⟩ + |1⟩)/√2 ⊗ (|0⟩ + |1⟩)/√2 = |+⟩ ⊗ |+⟩
Answer: This state is SEPARABLE (not entangled). It's the product of two |+⟩ states. 
import numpy as np
# The state
psi = np.array([1, 1, 1, 1]) / 2
# Check if separable using SVD
matrix = psi.reshape(2, 2)
U, S, Vh = np.linalg.svd(matrix)
rank = np.sum(S > 1e-10)
print(f"Rank: {rank}")  # Rank = 1 means separable
# Output: Rank: 1 (Separable!)

Exercise 3: CNOT Action

Problem: What is the result of applying CNOT to the state (|0⟩ + |1⟩)/√2 ⊗ |1⟩?

Solution:

First, expand the input state:

|ψ⟩ = (|0⟩ + |1⟩)/√2 ⊗ |1⟩ = (|01⟩ + |11⟩)/√2

Apply CNOT (control=first qubit, target=second qubit):

CNOT|01⟩ = |01⟩ (control is 0, no flip)
CNOT|11⟩ = |10⟩ (control is 1, target flips)
CNOT|ψ⟩ = (|01⟩ + |10⟩)/√2 = |Ψ+⟩
Answer: The result is the Bell state |Ψ+⟩!

Exercise 4: Measurement Probabilities

Problem: For the state |ψ⟩ = (|00⟩ + 2|01⟩ + |10⟩)/√6, calculate all measurement probabilities.

Solution:

The amplitudes are: α00 = 1/√6, α01 = 2/√6, α10 = 1/√6, α11 = 0

Probabilities are |amplitude|²:

P(00) = |1/√6|² = 1/6 ≈ 16.67%
P(01) = |2/√6|² = 4/6 = 2/3 ≈ 66.67%
P(10) = |1/√6|² = 1/6 ≈ 16.67%
P(11) = 0

Verification: 1/6 + 4/6 + 1/6 + 0 = 6/6 = 1 ✓

Exercise 5: Bell State Identification

Problem: Which Bell state has the property that measuring both qubits always gives opposite results?

Solution:

The four Bell states are:

  • |Φ+⟩ = (|00⟩ + |11⟩)/√2 → Same outcomes (00 or 11)
  • |Φ-⟩ = (|00⟩ - |11⟩)/√2 → Same outcomes (00 or 11)
  • |Ψ+⟩ = (|01⟩ + |10⟩)/√2 → Opposite outcomes (01 or 10)
  • |Ψ-⟩ = (|01⟩ - |10⟩)/√2 → Opposite outcomes (01 or 10)
Answer: Both |Ψ+⟩ and |Ψ-⟩ give opposite measurement results. These are called "anti-correlated" Bell states.

Exercise 6: Dimension Calculation

Problem: How many complex numbers are needed to fully describe a 7-qubit pure state?

Solution:

For n qubits, the Hilbert space dimension is 2n.

n = 7 → dim = 27 = 128
Answer: 128 complex numbers are needed to describe a 7-qubit pure state.

Note: Due to normalization and global phase freedom, only 2×128 - 2 = 254 real parameters are actually independent.

Exercise 7: Creating Entanglement

Problem: Starting from |01⟩, what gates should you apply to create the Bell state |Ψ+⟩ = (|01⟩ + |10⟩)/√2?

Solution:

Apply Hadamard to the first qubit, then CNOT:

|01⟩ → (H ⊗ I)|01⟩ = (|0⟩ + |1⟩)/√2 ⊗ |1⟩ = (|01⟩ + |11⟩)/√2
→ CNOT(|01⟩ + |11⟩)/√2 = (|01⟩ + |10⟩)/√2 = |Ψ+⟩
Answer: Apply H ⊗ I followed by CNOT. This is the same circuit as for creating |Φ+⟩ from |00⟩!

Exercise 8: Partial Trace

Problem: What is the reduced density matrix ρA for the state |ψ⟩ = |01⟩?

Solution:

For the product state |01⟩ = |0⟩A ⊗ |1⟩B:

ρAB = |01⟩⟨01|

The partial trace over B gives:

ρA = TrB(|01⟩⟨01|) = |0⟩⟨0|
Answer: ρA = |0⟩⟨0| = [[1, 0], [0, 0]]

This is a pure state, as expected for a product state.
